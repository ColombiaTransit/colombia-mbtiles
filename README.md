# Colombia MBTiles Generator

This is a bunch of github actions to help generate mbtiles files with [Tilemaker](https://github.com/systemed/tilemaker) and [Planetiler](https://github.com/onthegomap/planetiler)

### Coverage
- Covers Colombia
